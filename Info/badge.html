Badge

Badge adalah fitur atau sistem penghargaan yang ada di bot DCFinity.

Cara Mendapatkan Badge:

Untuk mendapatkan badge, kamu harus menyelesaikan sebuah misi. Setiap misi memiliki badge yang berbeda. Misinya bisa kamu lihat di dalam 'daftar badge' yang ada di bagian paling bawah halaman ini.

Cara Mengambil (Claim) Badge:

Di dalam bot DCFinity, kami tidak menyediakan sistem claim otomatis. Jadi, jika pengguna merasa dirinya menyelesaikan sebuah misi, maka pengguna bisa melakukan permintaan/request kepada developer untuk memasukan suatu badge ke akunnya. Untuk melakukan permintaan, pengguna bisa menggunakan command { /getbadge }.

Sistem Pengambilan Badge:

Dikarenakan kami tidak menggunakan sistem claim otomatis, maka setelah pengguna menggunakan command { /getbadge }, badge tidak akan otomatis dimiliki pengguna. Namun sebaliknya, kami menggunakan cara manual untuk memasukan badge ke dalam akun pengguna. Jadi, setelah pengguna menggunakan command { /getbadge }, kami akan memasukan pengguna kedalam antrian. Lalu setiap pengguna yang masuk kedalam antrian akan diperiksa apakah sudah memenuhi misi/syarat untuk mendapatkan badge atau belum. Pengguna bisa mendapatkan badge setidaknya paling lambat 7 hari setelah pengiriman permintaan. Jika lewat dari 7 hari dan pengguna belum mendapatkan badge, maka pengguna tidak dianggap menyelesaikan misi atau syarat tidak terpenuhi.

Peraturan/Info Tentang Badge:

- Badge yang dimiliki pengguna bersifat permanen.
- Tidak boleh melakukan spam saat menggunakan command { /getbadge }
- Pengguna bisa menghapus badge yang dimilikinya. Namun, badge yang sudah dihapus tidak bisa dikembalikan.
- Masuk ke support server bot jika ada pertanyaan/kesalahan dalam permintaan permeriksaan.
