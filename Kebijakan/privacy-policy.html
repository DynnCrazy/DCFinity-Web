Kebijakan Privasi

Privasi Anda penting bagi kami. Halaman ini menjelaskan bagaimana kami mengumpulkan, menggunakan, dan melindungi data Anda saat menggunakan layanan bot kami.

1. Data yang Kami Kumpulkan

1.1 Kami mengumpulkan data pengguna seperti username, userID, displayName, channelID, serverID.
1.2 Kami juga mengumpulkan data input atau interaksi yang Anda berikan kepada kami (baik dari slash option, button dan modal).

2. Penggunaan Data

2.1 Data yang Anda masukkan/diberikan kepada bot kami digunakan hanya untuk menjalankan perintah yang diminta melalui layanan bot kami.
2.2 Setelah perintah berhasil dieksekusi, data tersebut akan segera dihapus dan tidak akan disimpan dalam server atau database apapun.
2.3 Pengecualian ada pada interaksi like/dislike pada command {/meme}. Saat Anda like atau dislike, data tersebut akan masuk kedalam channel private yang ada di server discord kami. Data yang disimpan hanya username, userID, dan jenis interaksi nya (suka atau tidak suka).
2.4 Pengecualian juga ada pada command {/chat}. Pada command ini, kami menyimpan data Anda untuk sementara waktu. Data Anda dipastikan hilang jika pada command {/globla-chat}, nama Anda tidak ada. Selain itu, data Anda juga bakalan otomatis hilang jika bot Offline (walaupun cuma 1 detik)

3. Keamanan Data

3.1 Kami memastikan bahwa data Anda aman selama proses eksekusi perintah.

4. Pembagian Data

4.1 Kami tidak akan membagikan data Anda jika tidak mendapatkan ijin dari Anda.
4.1 Kami dapat memberikan data Anda untuk kepentingan hukum tanpa ijin dari Anda namun dengan sepengetahuan Anda.

5. Perubahan Kebijakan Privasi

5.1 Kebijakan Privasi ini dapat diperbarui sewaktu-waktu. Kami menyarankan pengguna untuk secara berkala memeriksa halaman ini untuk mengetahui perubahan terbaru.

Hubungi Kami
